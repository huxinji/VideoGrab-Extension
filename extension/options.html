📦 安装指南

VideoGrab Pro 需要配合本地服务运行。

🪟 Windows
🍎 macOS
🐧 Linux
1
下载安装包
下载 server 文件夹中的所有文件到本地
2
安装 Python 3.8+
如果已安装可跳过。
点击复制 # https://www.python.org/downloads/
3
运行安装脚本
双击运行 install_windows.bat
点击复制 install_windows.bat
4
或者手动安装
点击复制 pip install -r requirements.txt python server.py
1
安装 Homebrew
点击复制 /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2
安装 Python 和 ffmpeg
点击复制 brew install python ffmpeg
3
下载并启动服务
点击复制 chmod +x install_mac_linux.sh && ./install_mac_linux.sh
